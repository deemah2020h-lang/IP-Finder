# IP Address Lookup Tool 🔍

A simple Java application that runs from the command line and retrieves and displays information about IP addresses.

## 📌 Project Concept
This project takes the IP address as input and returns basic information such as location and network details using an external API.


## ✨ Features
- IP Address Lookup
- Country and City Information
- Internet Service Provider (ISP) Details
- ASN Information (if available via API)
- Simple Console Output

## 🛠️ Technologies Used
- Java
- HTTP Requests (API Integration)
- JSON Parsing (e.g., org.json or Gson)

## 🚀 How It Works
1. The user enters the IP address in the console.

2. Java sends a request to the IP Geolocation API.

3. The response is returned in JSON format.

4. The data is parsed and displayed in the command window.

## 💡 Project Objective
To provide a simple and fast tool for visually analyzing IP addresses in an easy-to-use format.
